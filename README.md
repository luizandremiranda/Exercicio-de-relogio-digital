# 🕒 Relógio Digital

Um relógio digital desenvolvido com **HTML**, **CSS** e **JavaScript** que exibe a hora atual em tempo real no formato **12 horas (AM/PM)**.

## 📷 Demonstração

O projeto apresenta:

* Atualização automática da hora a cada segundo.
* Exibição de horas, minutos e segundos.
* Indicador de período (AM/PM).
* Plano de fundo em tela cheia.
* Layout simples, moderno e responsivo.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6)

## ⚙️ Como executar o projeto

1. Clone este repositório:

```bash
git clone https://github.com/SEU-USUARIO/relogio-digital.git
```

2. Entre na pasta do projeto:

```bash
cd relogio-digital
```

3. Abra o arquivo `index.html` em qualquer navegador.

Também é possível utilizar a extensão **Live Server** no Visual Studio Code para visualizar o projeto em tempo real.

## 📚 O que foi praticado

Durante o desenvolvimento deste projeto foram utilizados conceitos importantes de JavaScript, como:

* Manipulação do DOM
* Seleção de elementos com `getElementById()`
* Funções
* Objeto `Date`
* Métodos `getHours()`, `getMinutes()` e `getSeconds()`
* Operador ternário
* Formatação de números com `padStart()`
* Atualização automática utilizando `setInterval()`

Além disso, foram aplicados conceitos de HTML e CSS para criação da interface.
